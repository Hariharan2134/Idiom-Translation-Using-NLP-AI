# 🧠 AI Idiom Translation System

An AI-powered web application built with **Python** and **Streamlit** that translates English idioms into their meanings using a searchable idiom database. The project provides a simple, fast, and interactive interface for students, language learners, and professionals to understand English idioms with ease.

---

## 📌 Features

- 🔍 Search English idioms instantly
- 📖 View idiom meanings and explanations
- ⚡ Fast and lightweight Streamlit interface
- 📊 Excel-based idiom database
- 💻 Easy to deploy and customize
- 🎯 Beginner-friendly user interface

---

## 🛠️ Tech Stack

- Python 3.x
- Streamlit
- Pandas
- OpenPyXL
- Excel (.xlsx)

## 🚀 Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/AI-Idiom-Translation-System.git
```

### 2️⃣ Navigate to the project folder

```bash
cd AI-Idiom-Translation-System
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will start locally and open in your browser.

---

## 📖 How It Works

1. Launch the Streamlit application.
2. Enter an English idiom in the search box.
3. The system searches the idiom database.
4. The corresponding meaning is displayed instantly.
